# TG Shop Bot

Телеграм-бот интернет-магазина на aiogram 3 с каталогом, корзиной, заказами, ролями админов и встроенным workflow оплаты с чеками.

## Возможности

### Для клиента
- Каталог товаров с категориями
- Корзина и оформление заказа
- Пошаговое оформление: Имя -> Фамилия -> Отчество -> Телефон -> Город -> Отделение Новой почты
- Выбор доступного способа оплаты
- Отправка чека (фото или PDF) для предоплатных методов
- История заказов по разделам: Новые, В работе, Архив

### Для администратора
- Управление товарами: добавление, редактирование, удаление
- Управление категориями
- Просмотр и изменение статусов заказов
- Подтверждение/отклонение чеков
- Кнопка просмотра чека прямо из заказа
- Авто-перевод заказа в статус Оплачен при подтверждении чека
- Поиск заказов по чеку (на проверке, подтвержден, отклонен, есть/нет)
- Настройка методов оплаты (карта, Apple Pay, Google Pay, наложенный платеж on/off)
- Настройка шаблонов уведомлений
- Бэкап базы данных через кнопку в админ-настройках

## Технологии
- Python 3.10+
- aiogram 3
- SQLite

## Структура
- `satana.py` - точка входа
- `loader.py` - инициализация бота и Dispatcher
- `handlers/users/shop.py` - основной бизнес-flow магазина
- `handlers/users/user_menu.py` - главное меню и админ-настройки
- `handlers/users/shop_state.py` - FSM-состояния
- `keyboards/inline/shop_inline.py` - inline-клавиатуры магазина
- `utils/db_api/shop.py` - DB-слой магазина
- `data/config.py` - загрузка конфигурации

## Установка

1. Клонируйте репозиторий
2. Создайте и активируйте виртуальное окружение
3. Установите зависимости:

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python satana.py
```
